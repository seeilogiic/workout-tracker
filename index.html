GitHub Setup

To sync across devices, you'll need a GitHub Personal Access Token:

1

Create GitHub Token

Go to GitHub Settings → Developer Settings → Personal Access Tokens

Click "Generate new token (classic)" and select "gist" permissions

2

Enter Token

3

Sync Data

How It Works

  • Your workout data is stored as a private GitHub Gist
  • Data syncs automatically when you save workouts
  • Access your data from any device with the same token
  • Your data remains private and secure
